FAQ
The phone is not detected after connecting via USB ▼
Make sure iTunes or Apple Devices (from Microsoft Store) is installed and running. The Apple Mobile Device Service must be active. Try a different USB cable or port. On the iPhone, tap Trust when prompted. Then restart server.py.
Battery health shows 0% or incorrect value ▼
iSupply Scan reads battery health via Apple's IOKit diagnostic service (same as 3uTools). If the value is missing, the device may have an aftermarket battery without an Apple chip, or the iOS version restricts access. The app will fall back to showing the current charge level instead.
Storage capacity shows N/A ▼
Storage is read from Apple's com.apple.disk_usage domain. Make sure the device is fully unlocked and trusted. Reconnect the cable and try again. If the problem persists, check the server.py console for error details.
Two slots fill up for one phone ▼
This is caused by a race condition in USB detection. It has been fixed in the latest version of server.py. Please update to the newest release. If it still occurs, disconnect and reconnect the device.
How do I add more users / licenses? ▼
Open the Admin Panel at localhost:5000/admin, log in with your admin account, and go to the Users section. Click + New user and fill in the details. Each user needs a valid license type (Trial / Pro / Enterprise) and expiry date.
Can I run iSupply Scan on multiple computers? ▼
Yes — the server.py runs on each workstation independently. Each workstation needs its own license. Enterprise licenses allow unlimited installations within one company. Contact us at info@isupply.cz for multi-seat pricing.
